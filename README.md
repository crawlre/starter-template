# Starter Template

## Overview

このプロジェクトは、PDFLOCSS設計（FLOCSS + BEM）とモバイルファーストを意識したコーディングテンプレートです。
再利用性・保守性・拡張性を考慮した構成で作成しています。

## Tech Stack

- HTML5
- SCSS (Dart Sass)
- PDFLOCSS (FLOCSS + BEM)
- npm scripts
- Git / GitHub

## Setup

### 1. Install dependencies

```zsh
npm install
```

### 2. Start development (Sass watch)

```zsh
npm run sass
```

## Directory Structure

```zsh
src/
└─ scss/
├─ layout/
└─ style.scss
```
